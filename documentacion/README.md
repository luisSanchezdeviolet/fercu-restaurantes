# 📚 Documentación - Fercu Restaurante SaaS

Esta carpeta contiene toda la documentación necesaria para **instalar y configurar** el sistema.

## 📋 Guías de Instalación y Configuración

| Archivo | Descripción |
|---------|-------------|
| **01-STRIPE_SETUP.md** | Guía completa para configurar Stripe (pagos) |
| **02-SENDGRID_SETUP.md** | Guía para configurar SendGrid (emails) |
| **03-COMANDOS_STRIPE.md** | Comandos rápidos de Stripe |
| **04-GUIA_PRUEBAS_STRIPE.md** | Cómo probar pagos en modo test |
| **05-GESTION_SUSCRIPCION.md** | Gestión de suscripciones por parte del usuario |
| **06-SAAS_ADMIN.md** | Panel de administración del SaaS |
| **07-MULTI_TENANCY.md** | Arquitectura multi-tenancy del sistema |

## 📄 Roadmap y Estado del Proyecto
